norm_rank_transform() function

Getting a normal rank transformation function that can handle NA values from the following formula:

\(Y^t_i={\phi}^{-1}({{r_i-c}\over{N-2c+1}})\)

where \(r_i\) is the rank, \(N\) is the number of ranked samples, \(\phi^{-1}\) is the inverse normal funcion (equivalent to qnorm() in R). The original recommended value of \(c=3/8\) was per Beasley and Erickson, 2010, but we use \(c=0\). Our work appears to meet their sample size recommendations.

norm_rank_transform = function(x, c = 0) {
  stopifnot(is.numeric(x) & is.vector(x))
  x_noNA = which(!is.na(x))
  N = length(x_noNA)
  x[x_noNA] = qnorm((rank(x[x_noNA], ties.method = "average") - c) / (N - (2 * c) + 1))
  return(x)
}
